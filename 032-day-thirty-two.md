GIT REBASE.

I was tasked with rebasing a Git repo. Git rebase helps to keep a clean commit history. It reapplies a series of commits unto a different base commit.  That way, if you're working in a team and they made other changes while you were making yours, you can apply your changes ON TOP of their changes.

From the task here, the feature branch was rebased into  So, the feature branch now contains the master branch commits plus its own.
